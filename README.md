# MIT-samtle-game_public
창원대학교 게임개발동아리 M.I.T.를 소개하기 위해 제작한 게임입니다.

창원대학교 컴퓨터공학과 신입생들에게 동아리를 소개하는 행사(샘틀)이 끝난 후
게임개발동아리의 회원이 되는 과정을 보여주는 이야기입니다.

이미지와 사운드의 라이센스 문제로 인해 스크립트만을 별도로 올린 저장소입니다.

## 조작법
캐릭터 이동 : 방향키

달리기 : Shift

사물과 상호작용, 선택창 선택 : Space
